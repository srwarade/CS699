 FOLLOW THE FOLLOWING STEPS FOR CODE EXECUTION:
*********************************************************

 STEP1 : OPEN THE TERMINAL IN THE 'source' NAMED FOLDER.
 
 STEP2 : EXECUTE THE COMMAND 'python3 server.py' TO RUN THE SERVER.

 STEP3 : ENTER THE IP ADDRESS AND PORT NUMBER(HARD-CODED) TO LAUNCH THE WEBSITE.

 STEP4 : ENTER TEXT IN THE BOX ONLY IN 'HINDI' FONT FOR 'SYSTEM VOICE GENERATION' AND 'ENGLISH' FONT ALONG WITH 'HINDI' FONT CAN BE USED FOR 'API VOICE GENERATION'.

 STEP5 : CLICK ON THE BUTTON AVAILABLE ON THE WEBPAGE TO GENERATE VOICE THROUGH DIFFERENT MODES.
